# Fundamentos de SQL
## Que es SQL?
**SQL** (se pronuncia "ese-cu-ele" o "sequel") significa Structured Query Language, que en español es Lenguaje de Consulta Estructurado.

En palabras simples: Es el idioma que usamos para hablar con las bases de datos.

Imagina que una base de datos es una biblioteca gigante con millones de libros. Si quieres encontrar un dato específico, no vas a ir estante por estante; en su lugar, le gritas una instrucción al bibliotecario en su idioma (SQL) y él te trae exactamente lo que pediste en segundos.

Más específicamente, una base de datos es un sistema para gestionar datos que nos permite:

- Almacenar datos de forma confiable
- Recuperar información eficientemente
- Manipular datos sistemáticamente

## Para que sirve?
Casi todo lo que usas hoy tiene SQL detrás. Cuando entras a Netflix y ves tu lista de películas, una consulta de SQL le dijo a la base de datos: "Oye, tráeme las películas que este usuario empezó a ver y no ha terminado".

Con SQL puedes hacer cuatro cosas básicas (conocidas como CRUD):
- Create (Crear): Guardar datos nuevos.
- Read (Leer): Consultar o ver datos que ya existen.
- Update (Actualizar): Cambiar información vieja por nueva.
- Delete (Borrar): Eliminar lo que ya no sirve.

## Como se ve la informacion?
En SQL, la información se organiza en Tablas. Piensa en ellas como hojas de Excel, con filas y columnas.

**Columnas**: Son las categorías (Nombre, Edad, Correo).
**Filas**: Son los registros individuales (Juan, 25 años, juan@mail.com).
